# Snake Water Gun CLI Game

This is my first Python project.

It is a simple Snake Water Gun game that runs in the terminal.  
The user plays against the computer.

## Features
- Terminal based game
- Random computer moves
- Input validation
- Infinite gameplay until exit

## How to Run

Run the following command:

python game.py
